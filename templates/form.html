Нужно преобразовать написанный текст под нужный шрифт — “Times New Roman”
и нужный кегль — 14 пт (размер текста).
Найти их можно на панели инструментов во вкладке “Главная”.


Если выуже набрали текст, то выделите его перед преобразованием!

Нужно выделить весь текст и выровнить его “по ширине”.
Для этого воспользуемся инструментами выравнивания.
Найти их можно на панели инструментов во вкладке “Главная”.

    Инструменты:
  • выравнивание по левому краю
  • выравнивание по центру
  • выравнивание по правому краю
  • выравнивание по ширине

Применим необходимый, чтобы весь наш текст растянулся по всей ширине страницы.

Нужно добавить отступ для первой строки 1,25.
Так же задать межстрочный интервал 1,5 (полуторный).

Для этого выделяем весь текст и жмем по нему ПКМ (Правая кнопка мыши) открывается меню.
В нём открываем вкладку “Абзац”.

Если выуже набрали текст, то выделите его перед преобразованием!

    Инструменты:
  1. Выбираем "Вставка"
  2. Выбираем "Номер страницы"
  3. Выбираем "Внизу страницы"
  4. Выбираем "Простой номер 1"

Обратите внимание!
Титульный лист считается первым, но не нумеруется.
Для этого после выбора нумерации нужно установить галочку "Особый колонтитул для первой страницы".

Создание документа

Пример правильного оформления:

Книга:
Фамилия, И. О. (Год). Название книги. Место публикации: Издательство.

Статья:
Фамилия, И. О. (Год). Название статьи. Название журнала, том(номер), страницы.

Интернет-источник:
Фамилия, И. О. (Год). Название статьи. Название сайта. URL (дата обращения).
